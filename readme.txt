The task:
Write a test using the algorithm:
1) go to the onliner.by main page. Select the section "Catalog" -> "Computers and networks" -> "Accessories" -> "RAM"
2) Choose a Dell manufacturer
3) Select any two RAM modules from the result, mark them and go to the comparison.
4) As a result, display the name of the RAM module, which has the highest score in comparison. Criteria for comparison are taken online; where more lines are allocated, that module is better.

Write this test using Selenium, TestNG and PageObject.
